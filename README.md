# experiment2026

2026年度数理情報工学実験リポジトリ

## 注意

- :warning: このリポジトリは学期末に Public にする可能性があるので、API キーや個人情報など **公開されて困る情報は保存しないこと！**

## 実験の進め方

- この `README.md` は、プロジェクト概要や構成が固まってきたら適宜更新して構いません。
- `log.md` に各回の活動内容を追記し、その日にやったこと・決まったこと・次回までの ToDo を記録してください。

## リポジトリ構成

- `README.md`: このリポジトリの概要
- `log.md`: 各回の実験ログ
- `src/`: ソースコードや実装ファイルを置くディレクトリ
- `imgs/`: 写真、図、スクリーンショットなどを置くディレクトリ
- `docs/`: 発表資料や補足資料を置くディレクトリ

※ `src/`、`imgs/`、`docs/` は必要になったら作成してください。
